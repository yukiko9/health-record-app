# 命名清单（统一 `-` 规则）

## 规则
- 所有页面、组件、元素标识统一使用 `kebab-case`。
- 历史命名中的空格统一替换为 `-`，例如 `float tip` -> `float-tip`。
- 历史命名中的下划线统一替换为 `-`，例如 `enter-ui-old` -> `enter-ui`。
